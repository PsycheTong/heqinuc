使用于移动设备友好地交互地图

2019年4月8日 - 极地图正式发布

我公司推出的极地图服务是国内领先的一套开源 Javascript 库，主要针对数据映射、地图映射、大数据可视化、WebGIS综合应用、WebGL等应用领域，不管在移动端还是PC端都有良好的兼容性。

极地图架构设计基于软件工程低耦合、高聚合的原理，应用前端领域先进的模块化开发组件及es6特性，以致整个架构性能和可用性能够达到最佳。开发者完全可以根据具体的项目需求动态载入相关组件。同时API中还为您提供了很多的扩展插件，使您的mapping过程更加高效、丰富，同时具有良好的观赏性和可交互性。

在这里，我们在‘map'div中创建一个地图，添加我们选择的切片，然后在弹出窗口中添加带有一些文本的标记：

//创建map对象，通过HTML中地图容器的ID获取该元素 var map = new CMMap.Map("mapcontent", { center: new CMMap.LngLat(116.489668,40.003576), zoom:15, animateEnable:false, zoomEnable:false }); //创建地图标记 var marker = new CMMap.Marker({ position: new CMMap.LngLat(116.489668,40.003506), icon: "img/other/position.gif", }); //在地图中添加ToolBar插件 map.plugin(["CMMap.ToolBar"], function() { toolBar = new CMMap.ToolBar(); }); //加载比例尺插件 map.plugin(["CMMap.Scale"], function() { scale = new CMMap.Scale(); }); //将标记和控件加入到地图中 marker.setMap(map); map.addControl(scale); map.addControl(toolBar);

特性

极地图专注于以最精简的代码实现基础功能的完美运作。

引入即用的图层

  • 多种地图服务
  • 多种标记、窗口
  • 矢量图层
  • 图像叠加
  • 支持GeoJSON格式

性能优化

  • 移动端提供硬件加速，体验效果更加流畅
  • 引入css3新特性，使动画效果得到加强
  • 加入智能点、线、面、多边形渲染功能，能够动态简化渲染质量，以达到最快的加载速度
  • 使用模块化构建系统。可以根据您的需求引入对应组件
  • 优化后，已经消除移动端设备的延迟效果

视觉特征

  • 缩放、平移动画
  • 平铺、弹出、淡入淡出动画
  • 优质marker
  • 视网膜分辨率支持

自定义功能

  • 多种css3制作的控件，随心更换样式
  • 图像和HTML都通用的marker
  • 自定义图层、控件
  • 自定义地图头一个
  • 强大的OOP工具

互动功能

  • 惯性拖动平移
  • 滚动变焦
  • 移动设备上的多指触控
  • 缩放到指定区域
  • 键盘导航
  • 监听事件
  • 标记拖动

浏览器支持

PC端

  • Chrome
  • Firefox
  • Safari 5+
  • Opera 12+
  • IE 7~11
  • Edge

移动端

  • 适用于Safari(IOS 7+)
  • Android浏览器 2.2+
  • Chrome移动版
  • Firefox for mobile
  • 用于Win8设备的IE 10+
