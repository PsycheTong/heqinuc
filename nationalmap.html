一个开源的JavaScript库

使用于移动设备友好地交互地图

2019年4月8日 - 极地图正式发布

极地图是领先的开源 Javascript 库，适用于移动设备友好的交互式地图。它只需要大约 38kb 的 JS ，它具有大多数开发人员所需的所有映射功能。

极地图的设计考虑了简单性，性能和可用性．它的工作原理有效地跨所有主流桌面和移动平台，可以有很多的扩展插件，有一个美丽的，易于使用和良好记录的 Apl 和一个简单的，可读的源代码，这是一个欢乐的贡献来。

特性

极地图专注于以最精简的代码实现基础功能的完美运作。

引入即用的图层

  • 多种地图服务
  • 多种标记、窗口
  • 矢量图层
  • 图像叠加
  • 支持GeoJSON格式

性能优化

  • 移动端提供硬件加速，体验效果更加流畅
  • 引入css3新特性，使动画效果得到加强
  • 加入智能点、线、面、多边形渲染功能，能够动态简化渲染质量，以达到最快的加载速度
  • 使用模块化构建系统。可以根据您的需求引入对应组件
  • 优化后，已经消除移动端设备的延迟效果

视觉特征

  • 缩放、平移动画
  • 平铺、弹出、淡入淡出动画
  • 优质marker
  • 视网膜分辨率支持

自定义功能

  • 多种css3制作的控件，随心更换样式
  • 图像和HTML都通用的marker
  • 自定义图层、控件
  • 自定义地图头一个
  • 强大的OOP工具

互动功能

  • 惯性拖动平移
  • 滚动变焦
  • 移动设备上的多指触控
  • 缩放到指定区域
  • 键盘导航
  • 监听事件
  • 标记拖动

浏览器支持

PC端

  • Chrome
  • Firefox
  • Safari 5+
  • Opera 12+
  • IE 7~11
  • Edge

移动端

  • 适用于Safari(IOS 7+)
  • Android浏览器 2.2+
  • Chrome移动版
  • Firefox for mobile
  • 用于Win8设备的IE 10+
